# hc-push-notification-plugin

FCM and APNs push notification support for the Apito API platform.

## Features

- Firebase Cloud Messaging (FCM)
- Apple Push Notification service (APNs)
- Topic-based notifications
- Device-targeted notifications

## Installation

```bash
go build -o hc-push-notification-plugin .
```
